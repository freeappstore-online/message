# Message

Private relay messenger. End-to-end, no server storage. Your messages live only on your device.

**Category:** Social  
**Live:** [message.freeappstore.online](https://message.freeappstore.online)

## Development

```bash
pnpm install
pnpm dev
```

## Deploy

Push to main — auto-deploys to R2 via GitHub Actions.
